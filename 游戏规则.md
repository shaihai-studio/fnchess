# 函数棋游戏规则

## 📖 游戏概述

函数棋是一款融合数学函数与策略对战的益智游戏。玩家需要通过构造函数图像，使其穿过目标格子来得分，同时避开禁止区域。

---

## 🎮 基本玩法

### 核心机制
1. **构造函数**：输入数学函数表达式（如 `x^2`、`sin(x)`、`tan(x)`）
2. **绘制图像**：系统自动在坐标系中绘制函数曲线
3. **碰撞检测**：检测函数图像是否穿过目标格子
4. **得分判定**：成功穿过目标格得分，进入禁止区则违规

### 游戏目标
- 在限定回合内获得最高分数
- 策略性地选择函数以命中多个目标格
- 避开禁止区域

---

## 📐 坐标系与棋盘

### 动态网格系统
- **初始范围**：-5 到 5（10×10 网格）
- **动态扩展**：随着回合数增加，坐标系范围自动扩大
- **缩放规则**：
  - 最小范围：5
  - 最大范围：50
  - 每次扩展步长：5

### 格子坐标
- 使用数学坐标系（向上为正，向右为正）
- 格子 `(x, y)` 表示：
  - x 范围：`[x, x+1]`
  - y 范围：`[y, y+1]`

## 🎯 得分规则

### 命中判定
函数图像必须**真正穿过**目标格子中心区域才算命中：
- ✅ **有效命中**：函数曲线距离格子中心垂直距离 < 0.5
- ❌ **擦边无效**：函数仅擦过格子边缘（距离 ≥ 0.5）

### 碰撞检测算法
采用**视觉检测（Pixel-based Sampling）**：
1. 在函数折线上高密度采样（每段21个点）
2. 检查采样点是否落在格子内部（带自适应容差）
3. 任一采样点满足条件即判定为命中

### 自适应容差
容差根据坐标系范围动态调整：
- 范围 = 5 时：容差 0.8px
- 范围 = 6 时：容差 0.667px（0.8 × 5/6）
- 范围 = 10 时：容差 0.4px
- 公式：`容差 = 0.8 × 5 / 当前范围`

---

## 🚫 禁止区规则

### 启用时机
- **第 5 回合起**开始出现禁止区
- 前 4 回合无禁止区

### 违规判定
- 函数图像进入任何禁止区格子 → 该回合违规
- 违规回合不得分

### 策略意义
- 需要在命中目标和避开禁止区之间找到平衡
- 高难度下禁止区布局更具挑战性

## 🧮 支持的函数类型

### 基础函数
| 函数 | 示例 | 说明 |
|------|------|------|
| 一次函数 | `x`、`2*x+1` | 直线 |
| 二次函数 | `x^2`、`x^2-3` | 抛物线 |
| 三次函数 | `x^3`、`2*x^3-x` | 三次曲线 |
| 高次函数 | `x^4`、`x^5` | 高次幂函数 |

### 三角函数
| 函数 | 示例 | 断点位置 |
|------|------|---------|
| 正弦 | `sin(x)` | 无断点 |
| 余弦 | `cos(x)` | 无断点 |
| 正切 | `tan(x)` | π/2 + nπ |
| 余切 | `cot(x)` | nπ |
| 正割 | `sec(x)` | π/2 + nπ |
| 余割 | `csc(x)` | nπ |

### 其他函数
| 函数 | 示例 | 定义域 |
|------|------|--------|
| 自然对数 | `ln(x)` | x > 0 |
| 常用对数 | `log(x)` | x > 0 |
| 指数函数 | `exp(x)`、`2^x` | 全体实数 |
| 平方根 | `sqrt(x)` | x ≥ 0 |
| 绝对值 | `abs(x)` | 全体实数 |

### 复合函数
支持函数组合和运算：
- `sin(x) + cos(x)`
- `x^2 * sin(x)`
- `ln(x^2 + 1)`
- `tan(x) / x`

## 🎨 函数图像渲染

### Desmos 风格绘制
采用专业数学工具的渲染逻辑：

1. **符号分析断点**
   - 预先识别函数的所有断点位置
   - tan(x)：π/2 + nπ
   - cot(x)：nπ
   - ln(x)：x ≤ 0

2. **分段采样**
   - 在断点之间独立采样
   - 避免跨越断点连线

3. **Y 值限制**
   - 限制采样点的 y 值范围
   - 防止延伸到屏幕外的线条

4. **分段绘制**
   - 每个连续区间独立绘制
   - 断点处不连线，消除竖线

### 动画效果
- 函数图像逐步绘制（动画效果）
- 绘制速度根据坐标系范围自适应调整
- 范围越大，绘制越快

## 🤖 AI 对手（Summa）

### 难度等级

#### 普通难度
- 使用简单函数（一次、二次）
- 基础策略，偶尔失误

#### 困难难度
- 使用中等复杂度函数（三角函数、分式）
- 三次函数使用整数参数
- 更精准的目标定位

#### 专家难度
- 使用复杂函数（高次、复合函数）
- **强制多目标穿透策略**：一箭双雕
- 最优策略选择

### AI 能力
- **参数反推**：根据目标格反推函数参数
- **多目标规划**：一个函数命中多个目标
- **禁止区规避**：智能避开禁止区域

## 🎯 策略建议

### 基础策略
1. **观察目标**：先看目标格位置，再选择函数
2. **利用对称性**：奇函数、偶函数可以覆盖对称区域
3. **函数平移**：通过加减常数调整图像位置

### 进阶技巧
1. **多目标穿透**：一个函数穿过多个目标格
2. **陡峭函数**：tan(x) 等陡坡函数适合垂直方向的目标
3. **高次函数**：x^3、x^5 等可以弯曲绕过禁止区
4. **复合函数**：组合多个函数精确控制曲线形状

### 应对困难模式
- **同一列目标**：使用陡坡函数（tan、高次函数、大系数）
- **小数点锁定**：使用整数参数的三次函数
- **禁止区密集**：选择简单函数降低误触风险

## 🔧 常见问题

### Q: 为什么我的函数没有命中？
A: 检查以下几点：
1. 函数是否真正穿过格子中心（不是擦边）
2. 函数定义域是否包含目标格 x 范围
3. 是否有断点影响曲线连续性

### Q: tan(x) 为什么没有竖线？
A: 系统采用 Desmos 风格的分段绘制，自动识别并处理断点，不会出现错误的竖线。

### Q: 如何提高命中率？
A: 
1. 使用测试模式练习
2. 观察函数图像的走势
3. 学习利用函数的周期性和对称性

### Q: AI 为什么能命中多个目标？
A: AI 使用参数反推和多目标规划算法，可以计算出同时穿过多个目标的最优函数。
